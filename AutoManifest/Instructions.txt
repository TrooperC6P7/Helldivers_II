How to make:
-Install VSCode.
-Install py2exe extension.
-Code AutoManifest.py.
-Export as .exe.

How to use:
-Unzip individual patch files into folders in mod folder.
-Organize mod folder to match desired Arsenal layout.
	-The mod, option, and sub-option names are defined by the folder/sub-folder names.
	-The folder layout must match the desired layout in Arsenal: the main mod folder for the 
	mod, the option folders for the different assets that can be altered, and the sub-option 
	folders for the different ways those assets can be altered.
-Add and organize images.
	-Images can be anywhere in the mod folder/sub-folders, but their name must match the 
	folder name of the mod/option/sub-option they apply to. They must also be .jpg or .png.
-Run .exe.
-Enter mod folder directory (as prompted).
-Exit .exe (as prompted).
-Add guid (as directed in manifest).
-Add descriptions (as desired).
	-If the .exe is run on a mod folder that contains a pre-existing manifest, the version, 
	guid, and descriptions will be preserved, assuming the mod/option/sub-option names are 
	the same. Also keep in mind that the .exe is very picky about structural whitespace when 
	searching for these fields. In any case, the pre-existing manifest will be saved to a 
	.txt file, so you won't lose anything if the process doesn't work.
-Zip everything in the mod folder, including the new manifest.
-Profit.
